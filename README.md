# Sistema de Gestión Escolar API
Este proyecto es una API REST para la gestión de alumnos y consultas académicas.

## 🛠️ Tecnologías
- Node.js & Express
- MySQL
- Dotenv & CORS

## 🚀 Instalación
1. Clonar el proyecto.
2. Ejecutar `npm install`.
3. Configurar el archivo `.env`.
4. Ejecutar `npm run dev`
